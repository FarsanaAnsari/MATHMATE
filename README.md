# MathMate Laravel

MathMate Laravel is a web-based educational platform for managing schools, teachers, students, math games, and game scores.

## Features

- Admin login
- School management
- Teacher management
- Student management
- Teacher-school assignment
- Student-school assignment
- Math game management
- Game score tracking
- Admin, school, and teacher score views
- REST API routes with CORS support

## Tech Stack

- PHP 8.0+
- Laravel 9
- MySQL
- Composer
- Vite
- Laravel Sanctum
